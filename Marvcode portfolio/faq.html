FAQs

What is the dashboard about?

It is simply a platform used for selling codes and templates for fast development.

What are the daily views?

The daily views shows the number of times the dashboard page has been visited by all users

How many accounts can I create?

You can only create one account per web browser. For example if you used chrome to register you should still use chrome to login.

How can I delete my account?

On the profile page visit the settings page, then tap or click on the button delete account. It will delete every credentials of yours.

What web browser is better to use?

Preferably chrome.

Can I change my profile name?

No you can't except you re-register. It is the same as deleting your account, only that it is going to update your account if you register again.
